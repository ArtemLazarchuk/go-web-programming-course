# Калькулятор палива (Go)

Простий веб-калькулятор для розрахунків палива, переписаний з Kotlin/Compose на Go.

## Запуск

```bash
go run main.go
```

Або збірка та запуск:

```bash
go build -o fuel-calc.exe .
./fuel-calc.exe
```

Відкрийте в браузері: http://localhost:8080

## Структура

- `main.go` — HTTP-сервер, обробка форм та логіка розрахунків
- `templates/index.html` — HTML-форма з полями для обох завдань

## Завдання 1

Розрахунок коефіцієнтів переходу, складу сухої/горючої маси та нижчої теплоти згоряння.

## Завдання 2

Розрахунок складу робочої маси мазуту та нижчої теплоти згоряння.
